Facturacion
  • Demo realizada con JSF 2.2, Primefaces 5, Eclipse Kepler R2, base de datos PostgreSQL 9.2 y EclipseLink 2.5 (JPA).
  • En la carpeta bdd dentro del proyecto se encuentra el respaldo de la base de datos facturacion que fue utilizada en esta demo.
  • En el manejo de productos se utilizan Actions con parametros, la cual es una nueva caracteristica de JSF 2.x y nos permite evitar binding con los DataTables.
  • Las Entities fueron generadas automaticamente y se personalizo la entidad FacturaCab. En esta clase se adiciono el parametro cascade=CascadeType.ALL en la relacion @OneToMany con FacturaDet para poder insertar en cascada (cabecera y detalles de factura) en una sola transaccion.
